# Natos Atlas
this is a World map mod for minecraft beta 1.7.3 there are currently StAPI and ModLoader ports.


## EARLY STAGES OF DEVELOPMENT
heads up this mod is at its infancy of development, its not feature complete and has had hardly any testing.
existing features and designs are likely to change constantly as im finding an identity for it.

## How to Install
If you already have a mc instance setup then all you need todo is.
- Download either the StAPI or ModLoader version from releases
- put it in the mods folder.

otherwise more detailed instructions will come in future.

## Known Issues
- Map scales strange with the GUI scale setting.
- Multiplayer maps buggy and experimental / its a side affect that they work.
- performance issues with updating map tiles.
- performance issues with saving map tiles.

please feel free to open an issue if you find any issues that are not listed here.
